# DougFirBiometrics.csv
# The biometrics of 25 Douglas firs from the White River Valley, Washington on 20-Apr-01
#  tree: a unique number for each tree
#  observer: student making the observations: Ingrid or Dylan
#  circ: the circumference at breast height (meters)
#  eyeht: height to the eye of the observer (meters)
#  horizdist: horizontal distance between observer and tree (meters)
#  angle: angle between observer and top of tree (degrees)
#  height: estimated height of tree (meters) using right-angle trigonometry
# Data from the QELP website but originally gathered by
#   Ingrid McNeely and Dylan Morgan, SCCC
